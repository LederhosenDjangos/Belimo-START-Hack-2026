# Belimo START Hack 2026

Leakage detection demo for Belimo actuator data using real-time telemetry and a simulated differential-pressure scenario.

## Project Goal

This project demonstrates a practical leakage response workflow for HVAC/data-center pipelines:

- Monitor actuator telemetry in real time.
- Simulate differential pressure (`鈭唒`) behavior.
- Compute pressure derivative (`d鈭唒/dt`).
- Trigger emergency valve closing when `d鈭唒/dt` leaves configured safety bounds.

## Business Case

Assumption: a sudden pressure drop indicates leakage.

Response strategy:

1. Track `d鈭唒/dt` against upper/lower bounds.
2. If derivative is outside bounds, immediately force valve setpoint to `0%`.
3. Keep visual evidence and export data for reporting.

## Data Flow

- Commands are written via `set_process_data(...)` to InfluxDB (`_process`).
- Telemetry is read via `get_measurement_data(...)` from InfluxDB (`measurements`).
- In the Streamlit app:
  - Real-time plot is fully selectable (`x` and `y`).
  - Leakage example simulates `鈭唒` over time.
  - `d鈭唒/dt` is computed from simulated `鈭唒`.
  - Out-of-bound derivative triggers emergency close.

## Main Demo Features (`demo/app.py`)

- Header: **Belimo Case**
- Section: **Leakage Detection: Derivative Bounds on Differential Pressure**
- Controls:
  - Normal valve setpoint (%)
  - Lower/upper bound for `d鈭唒/dt`
  - Start/Stop `d鈭唒/dt` example
  - Manual emergency close / reset
- Plots:
  - Real-time actuator plot (`x/y` selectable)
  - Actuator window: setpoint vs feedback during simulation
  - `鈭唒` over elapsed simulation time
  - `d鈭唒/dt` with bounds
- Export on stop:
  - Saves CSV files into `demo/assets/`

## Generated CSV Exports

When pressing **Stop d鈭唒/dt Example**, the app overwrites:

- `demo/assets/actuator_position_window.csv`
- `demo/assets/delta_p.csv`
- `demo/assets/delta_p_derivative_bounds.csv`

## Recreate PNG Plots from CSV

Script:

- `demo/assets/generate_plots_from_csv.py`

Run:

```powershell
python demo/assets/generate_plots_from_csv.py
```

This generates:

- `demo/assets/actuator_position_window.png`
- `demo/assets/delta_p.png`
- `demo/assets/delta_p_derivative_bounds.png`

## Run Locally (venv)

From repository root:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install streamlit pandas altair influxdb-client matplotlib
streamlit run demo/app.py
```

## Run with Docker

From `demo/`:

```powershell
docker build -t demo .
docker run --rm -it -p 8501:8501 -v ${PWD}:/work demo
```

Then open:

- `http://localhost:8501`

## Notes

- Bounds default to `-0.3` and `+0.3` (`bar/s`).
- The simulated `鈭唒` sequence includes a stable segment, sudden drop, and extended steady tail.
- Git-generated artifacts and generated CSV/PNG files are ignored via `.gitignore`.
